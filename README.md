# Air Billing · Self-Serve Roadmap

Interactive product roadmap — 10 months × 8 workstreams × 3 milestones.

Built with **React + Vite + dnd-kit**. Editable in the browser:
- **Edit mode** — drag cards between months, reorder within a column, edit text inline, add/remove items.
- **Hard dates** — pin a deadline on any card.
- **Persistence** — edits save to `localStorage`; Export / Import JSON to share or version.

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build      # outputs to dist/
npm run preview
```

## Deploy (GitHub Pages)
The build uses a relative base (`base: "./"`), so `dist/` works from any Pages subpath.
